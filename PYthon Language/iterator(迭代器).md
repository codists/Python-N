# 定义(what)

## 官方定义

Python 官方文档 glossary 章节对 iterator(生成器)的定义为( https://docs.python.org/3/glossary.html#term-iterator)：

> An object representing a stream of data. 

## 自定义

如果一个对象定义了  `__iter__() ` 方法和  `__next__()` 方法，那么这样的对象称为迭代器(iterator)。
