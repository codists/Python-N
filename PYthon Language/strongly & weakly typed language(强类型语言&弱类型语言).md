# 定义(what)

## 官方定义

strongly typed language(强类型语言)：

> In a strongly typed language, the type of a variable will not, at a minimum, be automatically changed behind the scenes for a particular use of the variable(摘自 Ronald T. Kneusel 的著作《Strange Code》).

强类型语言：Python，

weakly typed language(弱类型语言)：

> A weakly typed language will change data types implicitly in different situations(摘自 Ronald T. Kneusel 的著作《Strange Code》).

弱类型语言：PHP

# 实现(how)

## strongly typed language(强类型语言)：

```python
a = 2
b = "2"
print(a + b) # TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

## weakly typed language(弱类型语言)：

```php
<?php
$a = 2;
$b = "2";
echo $a + $b; // 4
?>
```
